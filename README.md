## Supported Versions

- **Minecraft**: 1.21.11
- **Loader**: Fabric

## Installation

1. Download the latest release from [Releases](../../releases)
2. Place the `.jar` file in your Minecraft `mods` folder
3. Launch your game and enjoy lag-free gameplay!

## Development

To set up the development environment:

1. Clone this repository
2. Run `./gradlew build` to compile the mod
3. Run `./gradlew runClient` to test in development environment

For more setup details, see the [Fabric documentation](https://docs.fabricmc.net/develop/getting-started/setting-up).
